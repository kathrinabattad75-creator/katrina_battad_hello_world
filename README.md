# Katrina Battad

Description
This project is a simple Flask web application created as part of Activity 3 in IS 312. The application demonstrates the basic setup of a Flask framework, route definition, and running a local development server. When accessed through a web browser, it displays a “Hello World + Katrina Battad!” message.

Framework Used
Flask – A lightweight Python web framework used for building web applications.

How to Run the Application Locally

Open the project in PyCharm.

Make sure Flask is installed in the Python interpreter.

Run the app.py file.

Open a web browser and go to http://127.0.0.1:5000.

The message “Hello World Katrina Battad!” should appear on the page.
